# nyc-taxi-analysis
NYC taxi tipping analysis project using 2023 Yellow Taxi and High Volume For-Hire Vehicle trip data, combined with weather and gasoline price data, to explore tipping patterns and model key factors affecting passenger tips.

- Name: `Yaojin Wu`

**Research Goal:** The goal of my research is the factors that influence amout of tip. This report is for taxi driver, to help them can get more tips from the passenger. 

**Timeline:** The timeline for the research area is 2023.

To run the pipeline, please visit the `notebooks` directory and run the files, there are four part in the file `code.ipynb`':
1. `download`: This part is to download the TLC trip record data into raw data folder.
2. `preprocess`: this part tell all the detail for preprocessing steps using the dataset in data folder.
3. `plot`: this part use the data after preprocessing to draw some plots and outputs the plots to the `plots` directory.
4. `model` : The part used for modelling and analysing the dataset by LM, GLM and Lasson.  

The report of the project please visit the `report` directry. There are two format of report .tex and .pdf. The .tex file is the code to write the report by latex. The output of the latex is .pdf file.
